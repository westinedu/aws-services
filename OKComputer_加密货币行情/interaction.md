# 加密货币行情网站交互设计

## 核心交互功能

### 1. 币种选择器与实时行情面板
- **位置**: 主页中央区域
- **功能**: 用户可以从下拉列表选择主流加密货币（BTC, ETH, BNB, XRP, ADA, DOGE, SOL, TRX, LTC, DOT等）
- **交互**: 选择币种后，实时显示当前价格、24小时涨跌幅、市值、交易量等关键数据
- **视觉反馈**: 价格变动时数字颜色变化（涨-绿色，跌-红色），配合微小的脉冲动画

### 2. 多币种价格对比工具
- **位置**: 主页下方对比区域
- **功能**: 用户可以同时选择2-4种加密货币进行价格走势对比
- **交互**: 
  - 币种选择框（最多4个）
  - 时间范围选择器（24小时、7天、30天、1年）
  - 对比图表实时更新显示
- **数据展示**: 使用ECharts创建多线图表，每条线代表不同币种，支持图例点击显示/隐藏

### 3. 交易所信息筛选器
- **位置**: 交易所页面
- **功能**: 筛选和排序全球主要加密货币交易所
- **交互**:
  - 按交易量排序（高到低）
  - 按成立年份筛选
  - 按支持币种数量筛选
  - 按交易费用范围筛选
- **结果展示**: 卡片式布局显示交易所信息，包括logo、名称、24小时交易量、支持币种数、手续费等

### 4. 市场分析工具
- **位置**: 市场分析页面
- **功能**: 提供多种数据可视化分析工具
- **交互**:
  - 市值分布饼图（显示各币种市值占比）
  - 价格相关性矩阵（显示不同币种间的价格相关性）
  - 交易量趋势图（显示市场整体活跃度）
  - 波动率分析（显示各币种价格波动情况）
- **用户操作**: 点击图表元素可查看详细数据，支持数据下载功能

## 交互流程设计

### 主页交互流程
1. 用户进入页面 → 显示BTC默认行情
2. 点击币种选择器 → 展开币种列表（带搜索功能）
3. 选择新币种 → 行情面板平滑过渡到新数据
4. 点击"添加到对比" → 币种添加到对比区域
5. 在对比区域选择时间范围 → 图表更新显示对应时间段数据

### 交易所页面交互流程
1. 页面加载 → 显示所有交易所信息
2. 使用筛选器 → 实时过滤和重新排序交易所列表
3. 点击交易所卡片 → 展开显示详细信息（支持币种、交易对、费用结构）
4. 点击"访问网站" → 新标签页打开交易所官网

### 市场分析页面交互流程
1. 选择分析类型 → 切换不同的数据可视化图表
2. 调整时间范围 → 图表数据相应更新
3. 悬停图表元素 → 显示详细数值和说明
4. 导出数据 → 下载当前图表数据为CSV格式

## 响应式设计考虑
- 移动端优化：币种选择器改为全屏弹窗
- 图表自适应：在小屏幕上简化图表元素
- 触摸友好：所有交互元素最小点击区域44px
- 加载状态：数据加载时显示骨架屏和加载动画

## 数据更新机制
- 模拟实时数据：每30秒自动更新价格数据
- 变化动画：价格变动时使用颜色闪烁和数字滚动效果
- 缓存策略：减少API调用频率，提升用户体验
